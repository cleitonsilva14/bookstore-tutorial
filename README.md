# Spring Boot JPA

Projeto de estudos de API (bookstore) Spring Boot JPA:

- Spring Boot 3 (3.3.4)
- Java 17
- PostgreSQL 16 (Docker)

Requests: **GET, POST e DELETE** !!!

E com relacionamentos:
- **@OneToMany**
- **@ManyToOne**
- **@ManyToMany** 
- **@OneToOne**


```bash
./mvnw spring-boot:run
```

App:
- controllers
- dto
- models
- repository
- services
